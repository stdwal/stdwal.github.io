#### 在终端中使用Sublime Text:

在终端中输入
```
sudo ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl
```
为subl创建链接


#### 编写markdown

按下```ctrl+shift+p```打开命令面板，输入```install```并选择```Package Control: install package```
在跳出的面板中输入```markdown```选择安装```MarkdownEditing```
